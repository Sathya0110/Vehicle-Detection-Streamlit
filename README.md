# Image Vehicle Detection System

Detect vehicles from uploaded images using YOLOv8.

## Features
- Upload multiple images
- High-accuracy detection using YOLOv8
- Shows bounding boxes and vehicle counts
- Lightweight: YOLO weights auto-download
- Clean Streamlit interface

## Usage
1. Deploy to [Streamlit Cloud](https://streamlit.io/cloud)
2. Upload images
3. See detection and counts

No Python installation needed.
